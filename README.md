# Memorial Purchase Infrastructure

Инфраструктура выбора, проектирования, оформления, производства, установки и последующего обслуживания памятников и мемориальных комплексов.

## Статус

**Discovery / Product definition. Реализация продукта не начата.**

До прохождения Product Gate запрещено создавать публичный сайт, выбирать окончательную архитектуру приложения или реализовывать бизнес-функции.

## Роли

- **Владелец продукта:** Андрей — бизнес-решения, приоритеты и окончательное утверждение продукта.
- **Менеджер проекта:** ChatGPT — исследование, декомпозиция, постановка задач, проверка отчётов и Pull Request, управление следующими шагами.
- **Исполнитель:** Codex — реализация только по утверждённой GitHub Issue, в отдельной ветке и через Pull Request.

## Базовое правило

GitHub — единственный источник истины. Задача не считается поставленной, если её нет в Issue. Работа не считается выполненной, если нет Pull Request, отчёта, проверок и обновлённой документации.
